## 1. Project Description
This is a secure web application developed using the Django framework. The system implements a **Secure Inventory Management System** with Role-Based Access Control (RBAC). The primary objective is to demonstrate defenses against OWASP Top 10 vulnerabilities, specifically focusing on injection attacks, broken access control, and security misconfiguration.
